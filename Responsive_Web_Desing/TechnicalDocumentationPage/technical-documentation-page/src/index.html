Que es C++

C++ es un lenguaje de programación de bajo nivel basado en C. Está caracterizado por incorporar los objetos y los strings, el tipo de variable que almacena textos. Un ejemplo de programa construido en C++ es google, el buscador.

  • C#
  • C
  • Java
  • JavaScript
  • Rust
Como empezar con C++

Para empezar hay que incluir algunas librerias y otras cosas imprescindibles

#include "iostream"
using namespace std;
int main()
{
return 0;
}

Estas lineas hacen que se pueda ejecutar el código, el return 0 indica que ahí se acaba el programa. La primera linea indica que librerias se han de utilizar

Definir variables

Definir variables es una parte fundamental de la programación, hay que hacelro así:

int nombreVariable = 23; //El int indica que el tipo de variable es un numero
nombreVariable = 2; //Cambiar una variable

Hay muchos tipos de variables: char(caracteres), float(decimales), strings(texto)....

Si pones el nombre de la variable y ; la definiras con valores basura

float nombre;
Cout y Cin

Cout se utiliza para imprimir valores por pantalla y, Cin se utiliza para coger valores por pantalla

Además hay lenguajes de programación escritos en C++ cout << "Inserte su edad"; //Saca por pantalla el texto
int edad = 0; //Define la variable con valor de 0
cin >> edad; //Recoge el valor que el usuario introduzca

También puedes usar cout para imprimir variables

If y Else

Así se hacen las condicionales de C++

if(7 > 4)
{
Soy mayor
}
else
{
Esto no puede pasar
}

También existe la orden de else if

if(7 > 4)
{
Soy mayor
}
else if (7 < 4)
{
Esto no puede pasar
}
